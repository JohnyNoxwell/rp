Qurrex
Терминал для торговли на Forex и криптобиржах
Web
Задача
Биржа для торговли криптовалютой
Решение
Разработана биржа по торговле криптовалютой с широким инструментарием, использует технологии DEX и CEX. В рамках биржи выпущена криптовалюта Qurrex.
Технологии
React
Javascript framework
CSS Modules
styles in-code
Mobx
storage framework
SCSS
styles pre-processing
Гибридная торговая площадка
Qurrex представляет собой многофункциональную гибридную торговую площадку, включающую несколько важных элементов: центральный узел, пользовательский элемент и блокчейн инфраструктура
Блокчейн
Блокчейн инфраструктура способна обеспечить решение вопросов в области надежного и безопасного хранения сведений о финансовых операциях. Инфраструктура позволяет исключить деятельность посредников в данной области.
Потребность
Qurrex подготовит инфраструктуру криптовалютной индустрии для классических и FX-брокеров, HFT, банков, поставщиков ликвидности и других институциональных игроков.
